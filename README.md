# Table 1 Reproduction — Lakbir et al. (npj Precision Oncology, 2025)

This repository contains the input manifests, metadata, and example download script
required to obtain the raw TCGA data used to recreate **Table 1** from:

> Tumor break load quantitates structural variant–associated genomic instability  
> with biological and clinical relevance across cancers (Lakbir S. et al., 2025)

Raw data **is not included**. Instead, `input_data/download.sh` provides instructions
to download all open-access files from GDC and TCGA PanCanAtlas.
